# microShell Sistemas Operativos
Microshell tipo Bash de la asignatura Sistemas Operativos

El microShell contiene estos módulos implementados:
  - Analizador de órdenes y argumentos
  - Visualizador de órdenes y argumentos
  - Implementación de redirección y tubos
  - Distinción entre procesos hijos y padres
  - Manejo de tablas de descriptores
  - Ordenes internas: cd y exit.
